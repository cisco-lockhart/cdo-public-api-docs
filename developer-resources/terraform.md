# Terraform Provider

The SCC Firewall Manager Terraform Provider can be used to  onboard and manage devices and other resources.
Download it from the [Terraform registry](https://registry.terraform.io/providers/CiscoDevNet/sccfm/latest/docs).

## Examples

View examples of how to use the Terraform provider on [GitHub](https://github.com/CiscoDevNet/terraform-provider-sccfm/tree/main/provider/examples)
